# Collector

Nossa aplicação tem por objetivo ajudar colecionadores em geral a listarem items de seu estoque, ou que tem um plano para comprar um colecionável. O aplicativo resolve o problema da desorganização e da dificuldade em acompanhar o inventário de coleções de forma simples e intuitiva.

## Funcionalidades

- Lista de desejados
- Lista de adquiridos (em estoque)
- Filtro por pesquisa
- Item personalisável (Foto, nome, preço)
